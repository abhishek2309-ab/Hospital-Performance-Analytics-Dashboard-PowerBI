# 🌟 Project Overview

The **Hospital Performance Analytics Dashboard** is a comprehensive Business Intelligence solution developed in **Power BI** to provide hospital administrators, healthcare managers, and decision-makers with a unified view of hospital performance.

The dashboard combines **Financial Analytics**, **Clinical Performance Monitoring**, **Patient Satisfaction Analysis**, **Insurance Insights**, and **Operational Metrics** into a modern and interactive reporting experience.
